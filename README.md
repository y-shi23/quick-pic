# Quick Pic - 快速图片处理工具

一个基于 TypeScript 的快速图片处理工具,可以为图片添加蒙版、调整透明度和模糊效果。

## 功能特性

- **蒙版效果**: 为图片添加自定义颜色的蒙版层,可调节透明度(0-100%)
- **透明度调整**: 调整图片整体透明度(0-100%)
- **模糊效果**: 添加高斯模糊效果(0-20px)
- **实时预览**: 所有调整实时显示在画布上
- **下载功能**: 支持下载处理后的图片为 PNG 格式

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

服务器将在 http://localhost:3000 启动

### 构建生产版本
```bash
npm run build
```

## 使用说明

1. 点击"选择图片"按钮上传图片
2. 使用左侧控制面板调整各项参数:
   - **蒙版透明度**: 控制蒙版的不透明程度
   - **蒙版颜色**: 选择蒙版的颜色
   - **图片透明度**: 调整原图的透明度
   - **模糊程度**: 设置图片的模糊程度
3. 点击"重置"按钮恢复所有设置
4. 点击"下载图片"保存处理后的图片
